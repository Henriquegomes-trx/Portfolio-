# Portfolio - Linguagem C
repositório destinado a projetos e trabalhos desemvolvidos em linguagem C
durante a graduação em análise e desenvolvimento de sistemas.

## Projeto em destaque:

### Sistema de gerenciamento de uma biblioteca com acervo de 20 livros
sistema desenvolvido com o objetivo de gerenciar, ordenar, listar, e pesquisar livros de um acervo.

### Funcionalidades
- Cadastro de livros.
- Ordenação de livros.
- Pesquisar por código.
- Verificação de códigos duplicados.

## Ferramentas utilizadas
- Linguagem C
- Estruturas ("struct")
- vetores
- Funções
- Lógica de programação
- Estruturas condicionais
- Estrutura de repetição
